Multi-Tool

Github Link

Multi-Tool is an experimental personal note-taking application (which also powers this site).

Features

Block Reference

Every block comes with a unique identifier that can be referenced individually. For example, click on Multi-Tool: Bi-Directional Linking to jump to the Usage section.

Bi-Directional Linking

Whenever the user includes a tag or internal link (referred to as Frontlink) within a particular block, the application will automatically generate a corresponding Backlink in the document that the frontlink is directed to. This backlink will point back to the aforementioned block.

This makes it easier to write notes on the backlinks themselves. Users may want to summarize/aggregate information from various backlinks sharing the same topic. For example, see Frontlink.

This also makes it trivial to assign a note to a dashboard document: just tag the name of the dashboard.
