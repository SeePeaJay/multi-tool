Multi-Tool

Github Link

Multi-Tool is an (experimental) personal note-taking application.

Features

Block Reference

Every block comes with a unique identifier that can be referenced individually.

Bi-Directional Linking

Whenever a user assigns a tag to a block, the application automatically creates a corresponding backlink on the referenced page. This backlink points back to the original block where the tag was added.

This makes it easy to summarize/aggregate information from various backlinks that share the same topic.
