Multi-Tool

Github Link

Multi-Tool is an experimental block-based note-taking application.

Features

Block Reference

Sometimes users may want to directly reference a particular block to provide additional context. This feature facilitates that need.

Bi-Directional Linking

Sometimes users may want to tag blocks that are important. For example, they may want to tag a block containing:

  • term definitions (in which case the tag can just be the term - see below example gif)
  • key points that are worth highlighting
  • questions that require further investigation

In such cases, the application generates a backlink in the page referenced by the tag. This backlink points back to the block where the tag was originally added.

Because these backlinks are designed to be physically part of the note (meaning they can be moved around freely in the page), users can organize them based on the topics they share and review them more effectively.
